Em uso

Manter a marca ativa no Instagram com consistência consome tempo: montar post, legenda, publicar no horário, repetir. Essa ferramenta tira o trabalho manual do caminho.

O problema

Postar com regularidade é o que mantém a marca viva — mas fazer tudo na mão, todo dia, não escala. Cada publicação vira uma sequência de passos repetitivos e sujeitos a esquecimento.

A solução

Uma ferramenta de linha de comando que publica direto na conta da BCM Tecnologias usando a Instagram Graph API — posts simples e carrosséis, com legenda, de forma programática e repetível.

Como funciona

  • Graph API: autenticação por token de sistema e publicação pelos endpoints oficiais da Meta.
  • Carrosséis: monta o container de mídia múltipla e publica em um comando.
  • Node.js puro: scripts testar e publicar, sem dependências pesadas.
  • Playwright: apoio para fluxos que a API ainda não cobre.

Resultado

Publicação confiável e repetível, que tira o trabalho manual do caminho e mantém a marca presente — com previsibilidade.
