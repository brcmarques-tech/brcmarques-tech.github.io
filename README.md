# brcmarques.dev

Portfólio pessoal de **Bruno Marques** — desenvolvedor full stack & builder de produtos com IA.

🔗 **Ao vivo:** https://brcmarques-tech.github.io

## Stack do site
HTML + CSS + JavaScript puro (zero frameworks, zero build). Hospedado no GitHub Pages.

- Tema dark editorial/cinematográfico
- Animações em vanilla JS: preloader, cursor custom magnético, smooth scroll, reveals por scroll, contadores, hero em canvas (constelação interativa)
- Totalmente responsivo e com `prefers-reduced-motion`
